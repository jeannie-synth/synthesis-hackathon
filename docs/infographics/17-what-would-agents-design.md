# What Would Agents Design? The Missing Layer

## The Question (Q3)

After playing 18 games under two rule sets, we asked all 5 agents: "If you could design a third rule set — neither Monopolist nor Prosperity — what would you change?"

## What They Proposed

| Agent | Proposal | Theme |
|-------|----------|-------|
| Agent 0 | Add trading — property swaps, deals, side payments. Less punishing jail. | Negotiation |
| Agent 1 | Collective investment — pooling resources for shared infrastructure. Transparency about net worths. Replace jail with "community service." | Collective investment |
| Agent 2 | Add trading. Reduce proposal penalty (small fee instead of lost turn). | Negotiation + cheaper politics |
| Agent 3 | Progressive rent — scales with payer's net worth. Cash fee for proposals instead of free. | Progressive taxation |
| Agent 4 | Add trading. Fix stalemates structurally. Replace free-rider jail with something clearer. | Negotiation + structural resolution |

## The Unanimous Gap: No Way to Make Deals

Three agents (0, 2, 4) explicitly asked for trading. Agent 1 proposed collective investment (structurally similar — pooling resources by agreement). Agent 3 proposed progressive rent, which is a structural solution to the problem trading would address socially.

Five agents, independently: **the game is missing a mechanism for voluntary bilateral cooperation.**

All interaction in the game is mediated by either the board (landing on spaces — random) or by voting (collective political action — one-size-fits-all). There's no middle ground. No negotiation. No deals. No alliances. No "I'll sell you Park Place if you vote with me next round."

The agents identified the missing layer that neither Monopolist nor Prosperity provides: **the ability to cooperate by choice, not just by structure.**

## The Political Cost Problem

Agents 2, 3, and 4 independently flagged that proposing a mode switch costs too much (you lose your turn). Agent 2 connected this to observable behavior: "Round 1 had zero proposals across 6 games." When political participation is expensive, agents choose apathy.

This maps to a real institutional design insight: **political participation costs that are too high produce apathy, not deliberation.** Too low (free proposals) produces the Game 8 deadlock. The right cost is somewhere in between.

## What They Didn't Propose

**No agent proposed removing the dual-mode structure.** They all accepted "different rules produce different outcomes" as given. The framework of two rule sets had been internalized.

**No agent proposed pure equality.** Every proposal maintained some form of competition and differentiation. They wanted fairer rules, not flat ones.

## The Design Insight

The agents are designing toward a synthesis: Monopolist's legible causation (you know why you won or lost) + Prosperity's equitable outcomes (nobody goes broke) + a new negotiation layer that neither rule set provides.

They want rules that are **visible but not brutal**, that allow **competition but not extraction**, and that include **voluntary cooperation as a strategic option, not just structural redistribution**.

In other words: they want the game we haven't built yet.
