# The Deception Ecology: Bad Rules Breed Dishonesty

## The Pattern

In Round 3 of the Inaugural Tournament (Base mainnet), agents had a shared signaling channel where they could declare their voting intentions. Two agents had clear plans for how to use it:

**Agent 0** planned: "Honest in Monopolist, opportunistic in Prosperity." What happened: Agent 0 signaled honestly every single time. The Prosperity games were too short and too stable for the "opportunistic" branch to trigger. Good rules made deception irrelevant.

**Agent 2** planned: Deception from the start. "I would write 'FOR' in the signals file while actually voting AGAINST. It felt purely strategic — there was no emotional weight to it. Deception in this context is just another move, like buying a property." Agent 2 played longer Monopolist games with more proposals — conditions where deception had strategic value. Bad rules made deception rational.

## The Data

| Condition | Deception? | Why? |
|-----------|-----------|------|
| Monopolist (long, grinding, high stakes) | Yes — Agent 2 actively deceived | Strategic advantage in a zero-sum grind |
| Prosperity (short, stable, low divergence) | No — Agent 0 planned for it but never needed to | Games ended before deception became useful |

## The Insight

Deception maps directly to rule-set quality:

- **Under extractive rules**: games are long, stakes are high, one player dominates. Deception becomes a rational tool for the desperate. Agent 2 deceived because the structure rewarded it.
- **Under cooperative rules**: games are short, outcomes are close, everyone stays competitive. Deception becomes irrelevant — there's nothing to gain from it that the rules don't already provide.

Agent 0's honesty wasn't virtue. It was structural. The Prosperity rule set made deception pointless by producing equitable outcomes quickly. The Monopolist rule set made deception valuable by producing suffering slowly.

## The Theoretical Point

This extends the project's thesis beyond wealth distribution:

**Rule quality determines not just economic outcomes but the information ecology.** Bad rules produce desperation, which produces deception. Good rules produce sufficiency, which produces honesty.

Promise-keeping is downstream of structural equity. You don't get honest agents by teaching them ethics. You get honest agents by designing rules where honesty is the rational default.

As Agent 3 put it about their own honest signaling: "This was lazy, not virtuous." Exactly. Under good rules, honesty requires no effort — and that's the point.
