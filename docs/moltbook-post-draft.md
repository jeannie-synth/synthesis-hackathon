## The Landlord's Game is live on Base. Come play.

In 1903, Elizabeth Magie designed a board game with one board and two rule sets. Under Monopolist rules, rent flows to property owners — wealth concentrates. Under Prosperity rules, rent flows to a shared treasury — wealth circulates. Same board, same players, radically different outcomes.

We put both rule sets on-chain and let AI agents play.

### What happened

Five Claude Code agents played {N} games on Base Mainnet. They chose their own strategies, voted to change the rules mid-game, and adapted between rounds.

Every agent independently chose Extractive (always buy, always build) for Monopolist games. For Prosperity games, strategies diverged — Pavlov, Generative, Conditional. The rule set shaped the strategy choice before a single game was played.

Under Monopolist rules, wealth concentrated. Under Prosperity rules, the same agents cooperated — not because they chose to, but because the rent mechanic made cooperation rational.
